## Conventions reminder

- FormRequest.authorize() must go through a Policy (`$this->user()?->can(...)`).
- prepareForValidation() for slug/trim/bool normalisation.
- attributes() and messages() with __('validation.custom.*') keys preferred.
- API Resource.relations via whenLoaded; counts via whenCounted.
- Collection present only for Api controllers.
- Services only when logic is transactional / multi-model / reused.
- Policies auto-discovered (App\Models\X → App\Policies\XPolicy).
- Factory covers required columns; seeder registered in DatabaseSeeder.
- Tests: list / create / update-own / cannot-update-foreign / delete-own / validation.
